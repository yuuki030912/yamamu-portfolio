最強キャラランキングの評価基準

このランキングは、総プレイ時間1200時間超の実戦データに基づいて作成しています。以下の基準で評価しています。

  • 対人戦（ランクマッチ）での実戦性能 を最重視
  • ビルドは正義ビルド（正義×テンション構成）を前提に評価
  • パッシブ継承・装備が完成した状態での評価
  • レアリティ（BASARA > HERO > レジェンダリー）による基礎ステータス差を考慮
Tierの見方：S = 環境最強・替えが利かない / A = 非常に強い・第一候補 / B = 十分実用的・編成次第で活躍 / C = 他に選択肢がある場合は優先度低め

レアリティ別の基本的な強さ

レアリティ 基礎ステータス チーム編成制限 入手難易度
BASARA 最高 1人まで HEROスピリット8枚 or バサラシード
HERO 高い 2人まで クロニクル周回・プレイヤーズユニバース
レジェンダリー 標準 制限なし 通常プレイで入手可能

理想のチーム編成（BASARA+HERO+レジェンダリー）

現環境の理想的なチーム構成はBASARA 1人 ＋ HERO 2人 ＋ レジェンダリー 8人です。正義ビルドを主軸に、チームパッシブの効果を最大化する編成を目指しましょう。

編成のポイント

  • BASARA枠（1人）：GKに使うのが最も費用対効果が高い。FWに使うのも有力
  • HERO枠（2人）：エースFWとエースMFに配置するのが鉄板
  • 正義ビルド統一：選手8〜9人＋監督＋マネージャーを正義ビルドで揃える（最大60%バフ）
  • テンション枠：2〜3人をテンションビルドにしてフォーカス勝利時のTP獲得を強化

S HERO 忍原来夏 単体性能が異常に高く、1人で試合を壊せるポテンシャル。フォーカスATが高くドリブル突破も強力

MFはフォーカスATDFのバランスが求められるポジション。攻守両方で活躍できるキャラが高評価です。異属性フォーカスATDFパッシブの継承が必須。

B レジェンダリー 灰崎凌兵 レジェンダリーの中ではフォーカス値が高め。正義ビルドが出やすく編成しやすい
MFのポイント：MFはフォーカスの攻守両方に関わるため、テクニック（AT/DF両方に影響）を重視した育成が鍵。パッシブ継承では「異属性フォーカスATDF」「同属性フォーカスATDF」を優先しましょう。

BASARA化の優先順位

BASARA化はマーケット「荒ぶる者たちの巣窟」でHEROスピリット8枚と交換するか、バサラシードを使用します。チームに1人しか編成できないため、誰をBASARA化するかは非常に重要な判断です。

BASARA化おすすめ優先順位

  1. GK（円堂守） — チームに1人しかいないGKはBASARA化の恩恵が最大。KP上昇で失点が大幅に減る
  2. エースFW（豪炎寺修也 等） — シュートATの底上げで得点力が劇的に向上。試合を決める力がある
  3. 司令塔MF（鬼道有人 等） — フォーカスATDFの上昇で中盤の制圧力が上がる
  4. 城壁DF（壁山塀吾郎 等） — 守備の要として堅実だが、GK/FWほどの劇的変化は少ない

BASARA化するとビルドタイプの自由変更が可能になり、「新たな可能性」で必殺技の付け替えもできるようになります。育成の自由度が格段に上がるため、長期的に使い続けるキャラをBASARA化しましょう。

バサラシードは誰に使うべき？

バサラシードはVer3.0.1で1個配布された超貴重アイテムです。HEROスピリット8枚分を肩代わりしてBASARA化できますが、一度使うと消費されやり直し不可。2個目以降の入手方法は現時点で限られています（公式「かなり頑張る必要あり」）。

バサラシード使用のおすすめ

  1. 円堂守（GK）が最安定。GK枠のBASARA化は誰でも恩恵を感じる
  2. 既にHEROスピリット8枚でGKをBASARA化済みなら、エースFWに使用
  3. 推しキャラに使うのもアリ — ただし対戦で使うキャラを優先するのが効率的
注意：バサラシードでのBASARA化は、どのレアリティからでも可能です。HERO Lv50まで育てなくてもBASARA化できるのが最大のメリット。ただし、同キャラ2人目のBASARA交換はバサラシードのみ可能な点にも注意。

ポジション別おすすめパッシブ継承

キャラの強さを最大限引き出すにはパッシブ継承が不可欠です。ポジションごとに最適なパッシブが異なるため、継承先と継承元を把握しておきましょう。

ポジション 最優先パッシブ 次点パッシブ
FW 敵陣での自身のシュートAT +8% 同ポジションシュートAT +3.5%、異属性フォーカスATDF
MF 異属性フォーカスATDF 同属性フォーカスATDF
DF 異属性フォーカスATDF 同属性フォーカスATDF
GK 異属性フォーカスATDF 同属性フォーカスATDF

パッシブ継承の注意点として、継承元キャラは消滅します。同キャラ・同ビルド制限はなく、1キャラから1個だけ継承可能です。BASARA・HERO・他チーム編成中のキャラは素材にできません。

ビルド別おすすめキャラ

ビルドタイプによって相性の良いキャラが変わります。現環境では正義ビルドが最強ですが、組み合わせビルドも主流になっています。

正義×テンション構成（最強・おすすめ）

Ver1.5.0以降、チームパッシブに上限が設けられたため、単一ビルド特化より複数ビルドの組み合わせが主流です。正義で基礎ステを上げつつ、テンションで獲得TP量を増やす安定と火力のいいとこ取り構成。

  • 正義ビルドの選手: 8〜9人（監督・マネージャーも正義で統一）
  • テンションビルドの選手: 2〜3人（マネージャーに付けると効率的）
  • 残りの枠は状況に応じてカウンターやラフプレーを混ぜる

正義×ラフプレー構成（上級者向け）

正義のバフを維持しつつ、ラフプレーのファウル率減少効果でダッシュ吹き飛ばしも使える攻撃的な構成。手動操作が前提のため上級者向け。

正義×カウンター構成（ロングシュート特化）

ボール奪取後のシュートATアップを活かしたロングシュート特化構成。カウンタービルドのDFからロングシュートを狙う戦術と相性抜群。
