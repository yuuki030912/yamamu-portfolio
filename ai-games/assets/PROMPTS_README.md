2. 各画像を**指定のファイル名**で保存（リネーム）
3. `C:/Users/yamau/Downloads/MyCluade/ai-games/assets/` に配置
4. ブラウザで `/ai-games/` を強制リロード → 自動で反映される（HTML側は既に対応済み）
5. SNSで共有するとOGPが表示される
